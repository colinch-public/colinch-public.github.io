고용보험에 가입 안 되어 있어도 받을 수 있는 지원이 생각보다 많다. 실업급여는 못 받지만, 내일배움카드 300만원짜리 훈련비 지원부터 국민취업지원제도 구직촉진수당까지 — 프리랜서·알바 신분으로도 신청 가능한 제도가 있다. 단, 본인이 "의무가입 대상인데 사업주가 안 넣어준 케이스"인지 먼저 확인해야 한다.

## 먼저 확인할 것 — 나는 의무가입 대상인가

2022년 이후 14개 직종의 특수고용직·노무제공자는 **사업주 의무 가입 대상**이다. 내가 이 직종에 해당하면 고용보험에 가입되어 있어야 정상이고, 안 되어 있으면 사업주 잘못이다.

| 의무가입 14개 직종 |
|---|
| 보험설계사 |
| 신용카드 모집인 |
| 대출모집인 |
| 학습지 교사 |
| 교육 교구 방문강사 |
| 택배기사 |
| 대여제품 방문 점검원 |
| 가전제품 배송·설치기사 |
| 방문판매원 |
| **IT 소프트웨어 프리랜서** |
| 골프장 캐디 |
| 관광통역안내사 |
| 어린이 통학버스 기사 |
| 화물차주 |

월 보수액이 **80만원 이상**이면 의무가입 대상이다. IT 프리랜서가 여기 포함된 건 의외로 모르는 사람이 많다.

**가입 안 되어 있으면 어떻게 하나:**  
근로복지공단(☎1588-0075)에 신고할 수 있다. 사업주가 미가입 상태였다면 소급 적용을 받을 수도 있다. 이 경우 실업급여 수급 자격이 생길 수 있으니 먼저 확인하는 게 맞다.

## 내일배움카드 — 300만원 훈련비 지원

고용보험과 관계없이 신청 가능한 대표적인 제도다. **국민내일배움카드**를 발급받으면 5년간 최대 **300만원**까지 직업훈련 비용을 정부가 대준다.

**신청 가능 대상:**
- 미취업자 (프리랜서, 구직 중인 알바)
- 주 15시간 미만 단시간 근로자
- 특수고용직·노무제공자
- 자영업자 (연 매출 1억 5천만원 미만)

취업 중인 재직자는 원칙적으로 신청 불가지만, 주 15시간 미만이거나 특수고용직이면 된다.

**훈련비 자기부담금:**  
전액 무료는 아니다. 훈련 과정에 따라 자기부담 **15~55%**가 발생한다. 다만 취업취약계층(실업자, 저소득층)은 자기부담이 0~15% 수준으로 낮아진다.

**신청 방법:**
1. [고용24(work.go.kr)](https://www.work24.go.kr) 접속
2. 구직신청 등록 (미취업자의 경우)
3. 내일배움카드 신청 → 심사 → 카드 수령
4. 원하는 훈련 과정 수강 신청

처음에 이게 뭔가 싶었는데, 코딩 부트캠프나 자격증 준비 강의에도 쓸 수 있어서 꽤 실용적이다.

![내일배움카드 직업훈련 지원](https://images.unsplash.com/photo-1516321318423-f06f85e504b3?w=800&q=80)

## 국민취업지원제도 — 구직촉진수당 최대 300만원

고용보험 없이도 신청 가능한 또 하나의 제도다. 일정 조건을 충족하면 구직 활동 기간 동안 **월 50만원씩 최대 6개월(300만원)**을 받을 수 있다.

이 제도는 1유형과 2유형으로 나뉘는데, 프리랜서·알바는 보통 **2유형**에 해당한다.

**2유형 자격 조건:**

| 항목 | 기준 |
|------|------|
| 나이 | 15~69세 |
| 소득 | 기준 중위소득 100% 이하 |
| 재산 | 4억원 이하 (2026년 기준) |
| 취업 경험 | 최근 2년 내 100일 또는 800시간 이상 |

2유형은 현금 지원보다는 취업지원 서비스(상담·훈련·일경험) 중심이지만, 취업활동비용 일부(최대 195만원)는 별도 지원된다.

1유형과 2유형 비교는 [국민취업지원제도 1유형 vs 2유형]({% post_url 2026-06-23-10-00-00-234555-national-employment-support-type1-type2 %}) 포스트에서 정리해뒀다.

**신청:**  
고용24(work.go.kr) → 국민취업지원제도 → 신청

## 프리랜서 종합소득세 신고는 별개 문제

고용보험 얘기를 하다 보면 세금 얘기가 섞이는데, 이건 따로 생각해야 한다. 프리랜서는 3.3% 원천징수 후 다음 해 5월 종합소득세 신고를 하면 환급받거나 추가 납부하게 된다.

이 부분은 [프리랜서 종합소득세 신고 방법]({% post_url 2026-06-18-10-00-00-172830-freelancer-comprehensive-income-tax-guide %}) 포스트에서 따로 다뤘다.

## 2026년 변화 — 소득 기반 전 국민 고용보험으로 간다

지금 정부가 고용보험 체계 자체를 뜯어고치고 있다. 기존의 "주 15시간 이상 근로" 기준을 폐지하고, **소득이 발생하면 대상자로 인정**하는 방식으로 바꾸는 작업이 진행 중이다.

아직 전면 시행은 아니지만, 특수고용직 의무가입 직종은 계속 늘어나고 있고 적용 범위가 넓어지고 있다. 올해 안에 추가 직종이 포함될 가능성도 있으니 고용24에서 주기적으로 확인하는 게 좋다.

## 이런 건 주의해야 한다

**내일배움카드 발급 후 바로 취업하면:**  
재직자 신분이 되면 발급 조건이 달라진다. 단, 이미 발급받은 카드는 계속 사용 가능하다.

**국민취업지원제도 참여 중 알바 수입이 생기면:**  
신고 의무가 있다. 숨기면 환수 대상이 된다. 수입이 생기면 담당 취업지원사에게 미리 알려야 한다.

**14개 직종인데 월 보수 80만원 미만이면:**  
의무가입 대상에서 제외된다. 이 경우 고용보험 혜택은 사실상 없다. 내일배움카드, 국민취업지원제도 쪽으로 방향을 바꾸는 게 현실적이다.
