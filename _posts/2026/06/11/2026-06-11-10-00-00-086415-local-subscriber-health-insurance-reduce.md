# 지역가입자 건강보험료 줄이는 방법 — 실제로 되는 것만

![재정 절약 계획을 세우는 모습](https://images.unsplash.com/photo-1554224154-26032ffc0d07?w=800&q=80)

지역가입자 건강보험료는 직장가입자와 달리 소득뿐 아니라 재산·자동차까지 반영해 부과된다. 같은 소득이어도 집이 있거나 차가 있으면 보험료가 훨씬 높게 나온다. 모르면 당하는 부과 구조라 알고 대응하는 게 필요하다.

## 보험료 부과 기준 (2026년)

| 항목 | 반영 여부 |
|------|-----------|
| 근로·사업·이자·배당 소득 | 포함 |
| 국민연금·공무원연금 | 포함 |
| 주택 (공시가격 기준) | 포함 |
| 토지 | 포함 |
| 자동차 (4,000만원 이상 또는 4년 이내) | 포함 |
| 전월세 보증금 | 간주임대료로 일부 반영 |

## 보험료를 낮추는 방법

### 1. 오래된 자동차 말소 또는 처분

자동차는 **4,000만원 이상**이거나 **차령 4년 이하**인 경우에 부과 대상이다. 해당되는 차를 처분하면 보험료가 즉시 줄어든다.

- 폐차나 매각 후 건강보험공단에 변경 신고
- 차가 없어졌는데도 반영이 안 되면 공단에 직접 확인 요청

### 2. 소득이 줄었다면 즉시 신고

건강보험 지역가입자 보험료는 **전년도 소득** 기준으로 부과된다. 소득이 줄었거나 사라졌어도 다음 연도 11월 정산 전까지는 기존 기준이 유지된다.

- 소득이 없어졌다면 공단에 직접 방문하거나 전화로 소득 감소 신고
- 근거 서류: 사직서, 폐업 신고서, 소득 없음 확인서 등
- 조정 승인 시 다음 달부터 낮은 보험료 적용 가능

### 3. 재산 감소 신고

건물·토지를 매각했거나 공시가격이 낮아진 경우 공단에 변경 신청을 해야 반영된다.

- 매각은 등기부등본으로 확인 가능하므로 대부분 자동 반영되지만, 지연될 수 있음
- 공단에 직접 신고하면 더 빠르게 반영됨

### 4. 가족 피부양자로 올라타기

배우자나 부모 중 직장가입자가 있다면 피부양자로 등록하는 게 가장 확실하다.

- 보험료를 아예 안 내도 됨
- 단, [피부양자 소득·재산 기준]({% post_url 2026-06-09-10-00-00-061725-health-insurance-dependent-registration-conditions %})을 충족해야 함

### 5. 연 소득 336만원 이하 시 최저 보험료 적용

소득과 재산이 일정 기준 이하면 최저 보험료만 낸다. 2026년 기준 최저 보험료는 **약 19,780원/월** 수준이다. 재산이 전혀 없고 소득이 매우 낮다면 이 수준으로 낮아진다.

## 보험료 모의계산 해보기

실제 내 보험료가 얼마인지, 조건 변경 시 얼마나 줄어드는지 직접 계산해볼 수 있다.

- [건강보험료 모의계산기 (nhis.or.kr)](https://www.nhis.or.kr/nhis/minwon/retrievePremCalcMain.do)
- 소득, 재산, 자동차 항목별로 입력하면 예상 보험료 산출 가능

## 이런 건 주의해야 한다

- 소득이 없다고 신고하면 공단이 국세청 자료와 대조한다. 실제로 소득이 있는데 없다고 하면 추후 소급 청구된다
- 지역 보험료는 매년 11월에 전년도 소득 기준으로 재산정된다
- 임의계속가입 중인 경우와 지역가입자 보험료를 비교해서 더 저렴한 걸 선택하면 된다
