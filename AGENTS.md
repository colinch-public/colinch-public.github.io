# Colinch 공공정보 경량 하네스

- 공통 규칙은 상위 `AGENTS.md`를 따른다. 자동 포스팅 중 `CLAUDE.md` 전체를 읽지 않는다.
- 언어: 한국어 직접체 `~다`. 복잡한 공공 정보를 신청자 관점에서 쉽게 쓴다.
- 주제: 정부지원금, 복지, 4대보험, 세금·절세, 청년지원, 생활행정.
- 반드시 날짜 기준을 적는다. 자격 조건, 신청 방법, 금액·계산 기준, 주의사항을 구체적으로 쓴다.
- 웹 검색: 1~2회로 최신 이슈를 찾고, 최종 기준은 공식 출처(정부24, 복지로, 고용24, 국세청, 지자체/부처)를 확인한다.
- SEO: 제도명, 연도, 조건·금액·신청 방법을 title·description·첫 문단에 넣는다.
- 태그 후보: `실업급여`, `고용보험`, `건강보험`, `국민연금`, `청년지원`, `소상공인지원`, `긴급복지`, `주거급여`, `국민취업지원`, `청년도약계좌`, `연말정산`, `종합소득세`, `전입신고`, `정부24`, `복지로`, `고용24`.
- 시각 자료: 자격 조건 표, 신청 절차 흐름도, 금액·기간 비교표, 준비서류 체크리스트를 우선한다.
- 불확실한 지원금 액수나 신청 기간은 추정하지 않는다. 확인 안 되면 “공고 확인 필요”라고 쓴다.
