## 주제 방향과 예시

이 블로그의 모든 포스트는 아래 다섯 가지 방향 중 하나에 해당해야 한다.

### 방향 1 — 정부 지원금·복지 신청 가이드

자격 조건, 신청 방법, 지급 금액을 단계별로 정리한다. 공식 사이트보다 읽기 쉽게.

```
주제 예시:
- "실업급여 신청 방법 완전 정리 — 자격 조건부터 수급까지"
- "청년 주거급여 신청 방법 — 조건·금액·서류 한 번에"
- "소상공인 지원금 2026 종류별 정리 — 어디서 얼마 받을 수 있나"
- "긴급복지지원 신청 방법 — 갑자기 생활이 어려워졌을 때"
- "국민취업지원제도 1유형 vs 2유형 — 나한테 맞는 건 어느 쪽인가"
```

### 방향 2 — 4대보험 실용 정보

건강보험·국민연금·고용보험·산재보험의 실제 계산법, 절약법, 자격 변동 처리 방법.

```
주제 예시:
- "건강보험 피부양자 등록 조건 2026 — 소득·재산 기준 정리"
- "국민연금 임의가입 하면 뭐가 좋은가 — 납부 예외자 기준도"
- "직장 퇴사 후 건강보험료 줄이는 방법 — 임의계속가입 vs 지역가입"
- "고용보험 가입 안 된 알바·프리랜서가 받을 수 있는 지원"
- "산재보험 신청 방법 — 이게 산재인지 모를 때 확인하는 법"
```

### 방향 3 — 세금·절세 실용 팁

직장인·프리랜서·소상공인이 세금을 덜 내거나 환급받는 방법.

```
주제 예시:
- "연말정산 환급 늘리는 방법 — 놓치기 쉬운 공제 항목 정리"
- "프리랜서 종합소득세 신고 방법 — 처음 하는 사람 기준"
- "개인사업자 절세 방법 — 합법적으로 세금 줄이는 항목들"
- "부양가족 인적공제 조건 — 누구까지 올릴 수 있나"
- "퇴직금 세금 계산법 — 실수령액이 얼마인지 미리 알고 싶다면"
```

### 방향 4 — 청년·취준생 대상 지원 정리

청년 특화 지원 제도를 한 곳에 모아 정리한다. 신청 기간, 자격, 금액 중심.

```
주제 예시:
- "청년도약계좌 신청 방법 — 조건·월 납입액·정부 기여금 계산"
- "청년 내일채움공제 — 중소기업 취직자가 3년 후 받는 돈"
- "청년 월세 지원 신청 — 지자체별 금액 다른 거 정리"
- "대학생 국가장학금 소득분위 계산법 — 내가 몇 분위인지 미리 알기"
- "취업성공패키지 vs 국민취업지원제도 — 2026 기준 뭐가 바뀌었나"
```

### 방향 5 — 생활 밀착 행정 정보

이사·차량·출산·사망·창업 등 인생 이벤트마다 처리해야 할 행정 절차 정리.

```
주제 예시:
- "전월세 계약 후 해야 할 것들 — 확정일자·전입신고 순서"
- "출산 후 받을 수 있는 혜택 — 산모지원·아동수당·육아휴직 한 번에"
- "자동차 폐차 방법 — 절차·비용·말소등록까지"
- "사업자 등록 방법 — 개인사업자 처음 내는 사람 기준"
- "주민등록증 분실 재발급 방법 — 온라인으로 되는지, 안 되면 어디 가는지"
```

## 포스트 선정 방법 (매 포스팅 시 실행)

```
1. WebSearch로 오늘 정부 정책·지원금 최신 뉴스 확인
   검색어 예시:
   - "정부 지원금 2026 신규"
   - "복지 정책 변경 오늘"
   - "청년 지원 2026 발표"
   - "소상공인 지원금 신청 시작"
   - "건강보험 기준 변경"
   → 오늘 막 발표되거나 신청이 시작된 제도가 있으면 최우선으로 다룬다
   → 뉴스에서 확인한 내용은 반드시 공식 출처(복지로·고용24·국세청)로 재확인 후 작성

2. 계절·시기 적합성 확인
   → 연말정산(11~1월), 국가장학금 신청(2월·8월), 청년도약계좌 신청 기간 등
   → 지금 신청 가능한 것을 우선

3. 시리즈 현황에서 다음 미작성 주제 확인
   → 최신 뉴스와 겹치는 시리즈 주제를 연결해 작성하면 더 효과적

4. 중복 확인
   grep -rl "주제키워드" _posts/ --include="*.md"

5. 공식 정보 출처에서 최신 기준 확인 후 작성
   → 복지로(bokjiro.go.kr), 고용24(work.go.kr), 국세청(nts.go.kr), 건강보험공단(nhis.or.kr)
   → 기준 변경이 잦으므로 반드시 최신 연도 기준 명시
   → 뉴스 기사는 참고용. 실제 조건·금액은 공식 공고문 기준으로 작성
```

## 글쓰기 스타일

### 언어 수준

- 행정 용어는 반드시 괄호 안에 풀어쓰기
  - "피부양자(건강보험료를 따로 내지 않고 가족에 올라탄 상태)"
  - "확정일자(임대차 계약서에 법적 날짜 도장을 받는 것)"
- 문장은 짧게. 조건이 많으면 표로 정리
- 숫자·금액·날짜는 굵게 강조

### 톤

- 같은 처지로 먼저 겪어본 사람이 알려주는 식
- "몰랐으면 못 받을 뻔했다", "이게 생각보다 복잡하더라"
- 공식 사이트 읽다가 포기한 경험 공감에서 시작
